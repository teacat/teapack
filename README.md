比起 JSON 還要更有效率、更節省頻寬的位元組資料傳遞方式。

# 這是什麼？

TeaPack 是基於位元組資料與 [vmihailenco/msgpack](https://github.com/vmihailenco/msgpack)（亦為 MsgPack）
